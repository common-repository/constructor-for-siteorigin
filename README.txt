Creates a Widget to display a Masonry, Infinite scroll, filterable loop of posts or whatever custom post type you have. Includes 1-20 columns layout.

== Description ==

This plugin adds a widget to your Wordpress site called Constructor Siteorigin, publish this widget when you are building a post or cpt with SiteOrigin Page Builder, it will show metadata set in widget options:

show metadata as:

1-Post Title
2.Post Excerpt
3.Post Author
4.Number of comments
5.Post categories
6.Post Featured Image


== Installation ==


1. Upload `constructor-siteorirign` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow usage instructions, just plublish the Constructor for SiteOgrigin Widget and set its options.

== Frequently Asked Questions ==

= Do this plugin work without SiteOrigin Page builder, let's say in a asimple/normal sidebar? =

Yes, it is intended to construct/build posts with SiteOrigin PageBuilder, but ill also show widget meta if it is published in a single view sidebar.
